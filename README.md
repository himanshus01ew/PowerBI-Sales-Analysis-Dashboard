# Sales Analysis Dashboard using Power BI

## Project Overview
This project analyzes sales data using Microsoft Power BI to uncover business insights related to sales performance, profitability, customer behavior, trends, forecasting, and category performance.

## Objectives
- Analyze sales and profit trends
- Identify top and bottom performing categories
- Study customer and regional behavior
- Detect anomalies and outliers
- Forecast future sales trends
- Support data-driven business decisions

## Tools Used
- Microsoft Power BI
- Power Query
- Microsoft Excel

## Dataset Fields
- Sales
- Profit
- Quantity
- Discount
- Category
- Sub-Category
- Region
- Segment
- Order Date

## Analysis Performed
### Data Preparation
- Data Cleaning
- Missing Value Analysis

### Business Analysis
- Average Value Analysis
- Category Distribution Analysis
- Top & Bottom Performing Categories
- Histogram Analysis
- Correlation Analysis
- Trend Analysis
- Seasonal Analysis
- Forecasting Analysis
- Outlier Detection
- Contribution Analysis
- Hierarchical Analysis
- Decomposition Tree Analysis

## Key Findings
- Technology category generated the highest sales.
- Some high-sales orders produced negative profit.
- Seasonal patterns affected sales performance.
- Discounts reduced profitability in several cases.
- Forecasting indicated future sales growth.

## Recommendations
- Reduce excessive discounting.
- Focus on high-performing categories.
- Monitor loss-making transactions.
- Improve low-performing regions.
- Use forecasting for future planning.
